0.4.0 "Renaissance"
Changes:
  • eihort will try to autoload the latest available Minecraft Jar file
  • New config option to let eihort ignore missing textures
  • New html based readme file
  • Added localisation (for now english and german)
  • Added quit button
  • Faster texture loading
  • Text is now displayed with the Minecraft font
  • World now starts at the Minecraft world time
  • Added mouse sensitivity scale (to reduce both at the same rate)
  • Added more on screen information (TBA)
  • Added the last played version to the world menu toolbox
  • Added "F3" to toggle on screen information
  • Added "I" as goto worldspawn
  • Update for biome colours
  • Move player here now saves the rotation
  • Fix for incorrect time stamps in screenshot names
  • Fix for a crash while taking a screenshot on drag & dropped worlds
  • Fixed transparent areas in screenshots
  • Fixed screenshots not beeing saved on Linux
  • Fix for linux message boxes
  • - New 1.9 blocks added (TBA)
  • - New 1.10 blocks added (TBA)
  • - New old blocks added (TBA)
  • - Updated existing blocks (TBA)
  • Update to SDL 2.0
  • Update for most libs (now 64-bit)
  • Test implementation of nvidia optimus
  • Added developer tools
  • Texture fixes (eg. for the cobblestone slab)
  • General bug fixes
