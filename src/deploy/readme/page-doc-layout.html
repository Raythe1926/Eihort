the default key layout
Movement:
Find more valid keys here. Usage without "SDLK_"
  • W = Forwards
  • S = Backwards
  • A = Left
  • D = Right
  • SPACE = Up
  • X = Down
  • CTRL LEFT & CTRL RIGHT = Decrease movement speed
  • SHIFT LEFT & SHIFT RIGHT = Increase movement speed
Setting:
  • [ & PAGEDOWN = Decrease view distance
  • ] & PAGEUP = Increase view distance
  • T = Add time
  • Y = Remove time
Eihort:
  • R = Reload world
  • L = Change light mode
  • N = Change moon phase
  • O = Go to defined position
  • P = Go to players position
  • I = Go to world spawn
  • F1 = Toggle user interface
  • F2 = Take screenshot
  • F3 = Toggle on screenshot information
  • ESC = Free mouse
  • M = Toggle world monitoring
  • ALT LEFT = Toggle button effect
Spline keys:
  • Q = Add spline point
  • 0 = Delete all spline points
  • 1 = Jump to first spline point
  • 2 = Do spline (direction first to last)
  • 3 = Jump to last spline point
  • 4 = Do spline (direction last to first)
  • E = Delete last added spline point
Alternate button effects:
  • ALT LEFT + P = Move player to the current position
  • ALT LEFT + I = Move spawn to the current position
  • ALT LEFT + F2 = Take screenshot with GUI
