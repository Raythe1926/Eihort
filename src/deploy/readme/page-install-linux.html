to install and use eihort follow these steps
Ubuntu:
  1. Open the "System Settings"
  2. Go to "Software & Updates"
  3. Enable "Community-maintained free and open source software (universe)"
  4. Click "Close" and agree to reload the cache
  5. Update or download and install libglew version 2.0 from here
    Select the 32-bit ("i386") or the 64-bit ("amd64") version and choose any download server
  6. Install eihort's .deb package
Ubuntu (command line):
  1. Enable "Community-maintained free and open source software (universe)"
    add-apt-repository universe
  2. Reload the cache
    apt-get update
  3. install libglew 1.10 (download as above)
    dpkg -i path/to/libglew.deb
  4. Install eihort's .deb package
    dpkg -i path/to/eihort/package.deb
  5. Install other dependencies
    apt-get -f install
