command line parameters to startup eihort

All argument values have to be provided in their Minecraft format. Use a dot "." for decimal numbers (example 4.1).
eihort will only process the provided arguments and will fill the missing arguments with the player's data. If eihort can't find any arguments it will show the world selection screen.

world path
  • To set the displayed world use -w or -world followed by the Minecraft world's path
  • If there is no -w or -world argument eihort will use the first passed argument as world path. If no regions can be found eihort will display an error message.
X, Y & Z coordinates
  • To set the player's X position use -x or -eyeX followed by the target value
  • To set the player's Y position use -y or -eyeY followed by the target value
  • To set the player's Z position use -z or -eyeZ followed by the target value
view direction
  • To set the player's vertical view direction -p or -pitch followed by the target value
  • To set the player's horizontal view direction -a, -azi or -azimuth followed by the target value
dimension
  • To set the displayed dimension use -d, -dim or -dimension followed by
    • - overworld, o or 0 for the Overworld dimension
    • - nether, n or -1 for the Nether dimension
    • - end, e or 1 for The End dimension
world time
  • To set the world time use -t or -time followed by the target value
view distance
  • To set the view distance use -v, -vd or -viewDistance followed by the target value
  • Even the view distance can be set to values outside the set range, values smaller than 0 will be ignored. Using the keyboard to change the view distance will revert it into the set value range (default 50 to 10'000 blocks)
VRAM
  • To set the Video RAM/GPU memory size use -vram followed by the target value in megabyte
  • The VRAM size has to be bigger than 0.

Example:
"/path/to/eihort" -world "/path/to/the/minecraft/world" -x 0.5 -y 1000 -z 0.5 -pitch 90 -azimuth 180 -dimension overworld -time 6000 -viewDistance 15000 -vram 2048
This will start eihort using world found in "/path/to/the/minecraft/world". The displayed dimension will be the overworld and it will be viewed from 0.5/0.5, 1000 blocks above bedrock, looking north and straight down. The time is high noon, the view distance is 15'000 blocks and eihort will use 2048 MB of your videocard's memory.
