values in the eihort.config file
language:
  • => The language eihort uses for text
  • eihort will automatically try to load the system language
    To load a specific language, remove the "--" before and insert the correct language shortcut. For example
    language = "de";
    for the german localisation
minecraft_path:
  • => The location of the minecraft folder on your hard disk drive
  • minecraft_path = The location of the ".minecraft" folder on your hard disk drive
  • eihort will automatically try to load the latest available Minecraft version from the standard location
    To load Minecraft from a specific path, replace the existing text with
    minecraft_path = "path/to/.minecraft/";
  • Use either a forward slash ("/") or a double backslash ("\\") between the folders
eihort_path:
  • => The location of the eihort folder on your hard disk drive
  • eihort_path = The location of the "eihort" folder on your hard disk drive
  • On some systems with access restricted folders eihort is not able to write in it's directory
    To load eihort from a specific path, replace the existing text with
    eihort_path = "path/to/eihort/";
  • Use either a forward slash ("/") or a double backslash ("\\") between the folders
asset_search_paths:
  • => The location of all of your asset/texture files. Earlier entries in the list have priority.
  • minecraft_path = The location of the ".minecraft" folder on your hard disk drive
  • Adding paths to the list:
  • minecraft_path .. "path/to/file";
  • "full/path/to/the/texture/file";
world_paths:
  • => The location of your saved world file(s) on your hard disk drive (usually */.minecraft/saves/*)
  • minecraft_path = The location of the ".minecraft" folder on your hard disk drive
  • The format is either "{ "PATH", R, G, B };"
  • PATH = The world path on your hard disk drive
  • R = The ammount of red on the map select folder (value for each from 0.00 to 1.00)
  • G = The ammount of green on the map select folder (value for each from 0.00 to 1.00)
  • B = The ammount of blue on the map select folder (value for each from 0.00 to 1.00)
  • or "{ "PATH", "#HEX-COLOUR" };"
  • PATH = The world path on your hard disk drive
  • #HEX-COLOUR = the colour HEX value (eg #2666CC for the default eihort-blue)
  • For world folder use either a forward slash ("/") or a double backslash ("\\") between the folders
resource pack:
  • => The location of an optional resource pack, eihort will load on startup
  • To load a resource pack, replace the existing text with
    minecraft_path .. "resourcepacks/my_favourite_resource_pack.zip";
  • minecraft_path = The location of the ".minecraft" folder on your hard disk drive
screenshot_path:
Find more valid variables here.
  • => The location where your screenshots will be saved in (usually */eihort/screenshots/*)
  • eihort_path = The location of the "eihort.exe" file on your hard disk drive
  • Use "%O" to include the world's name
screen options:
  • => Options for eihorts start screen
  • fullscreen = eihort will (true) or will not (false) start in full screen mode
  • screenwidth = set the width of eihort startup widndow
  • screenheight = set the height of eihort startup widndow
  • multisample = anti-aliasing (default = 4)
  • anisotropy = anisotropic filtering (default = 4)
  • fov = the degrees of the Field Of View (default = 77)
worldmenu info:
  • => Options for additional displayed info in the world menu
  • show_last_played_version = eihort will (true) or will not (false) display the world' last Minecraft version
info display:
  • => Options for additional displayed info
  • show_triangles = eihort will (true) or will not (false) display the ammount of triangles in a scene
  • show_vram_use = eihort will (true) or will not (false) display how much video card memory is used
  • show_region_file = eihort will (true) or will not (false) display where your current position is saved in
  • show_global_chunk = eihort will (true) or will not (false) display in which chuck you are in for the whole map
  • show_region_chunk = eihort will (true) or will not (false) display in which chuck you are in for the region file
  • show_view_angles = eihort will (true) or will not (false) display the pitch and azimuth view angles
speed change:
  • => Values the movement speed will be multiplied with
  • speedup_modifier = the default movement speed will be accelerated by the inserted value (default = 8)
  • slowdown_modifier = the default movement speed will be decelerated by the inserted value (default = 1/8)
key bindings:
  • => The default keys and their actions
  • see key layout for more information
mouse sensitivity:
  • => Option to adjust the mouse sensitivity in radians per pixel
  • mouse_sensitivity_scale = the mouse sensitivity in percent for the x- and the y-axis (default = 100.0)
  • mouse_sensitivity_x = the mouse sensitivity on the x-axis (default = 0.008)
  • mouse_sensitivity_y = the mouse sensitivity on the y-axis, set to negative to invert (default = 0.01)
movement speed:
  • => Option to adjust the movement speed in blocks per second
  • movement_speed = the speed on the map page in blocks per second (default = 50)
  • view_distance_speed = the speed of the view distance change in blocks per second (default = 50)
  • time_speed = the speed of the time change in hours per second (default = 3)
  • spline_speed = the speed of spline points per second (default = 0.4)
O-position:
  • => Option to adjust the place you get to when "O" is pressed
  • origin_x = X coordinate (default = 0)
  • origin_y = Y coordinate (default = 100)
  • origin_z = Z coordinate (default = -20)
  • origin_pitch = vertical view direction in radians, π/2 is up, - π/2 is down (default = -0.7)
  • origin_azimuth = horizintal view direction (default = 0)
world startup:
  • => Option to adjust eihort's world start up setting
  • use_world_time = eihort will (true) or will not (false) use the Minecraft world time as startup time
  • start_time = time on map startup, -1 and 1 are midnight, 0 is noon (default = 0)
  • start_view_distance = view distance on map startup (default = 500)
fps limit:
  • => Option to adjust the maximum frames per second
  • fps_limit = limit of the drwawn frames per second (default = 60)
comfort settings:
  • => Option to adjust if eihort should ask before performing a task using the toggle key
  • confirmToggledButtonUse = eihort will (true) or will not (false) ask before performing a task in general
  • confirmToggledButtonUseForMovedPlayer = eihort will (true) or will not (false) ask before moving the player
  • confirmToggledButtonUseForMovedSpawn = eihort will (true) or will not (false) ask before moving the spawn
tooltip settings:
  • => Option to adjust if eihort should display button tooltips
  • showShortcutButtonTooltip = eihort will (true) or will not (false) show tooltips in in general
  • showShortcutButtonTooltipForEOMenu = eihort will (true) or will not (false) show button tooltips in the extra options menu
  • showShortcutButtonTooltipForDevButtons = eihort will (true) or will not (false) show button tooltips for developer buttons
screenshots settings:
  • => Option to adjust screenshot settings
  • takeScreenshotAsIs = eihort will (true) or will not (false) screenshot what it renders (including GUI etc.)
advanced settings:
  • => Option to adjust advanced settings
  • qtree_leaf_size = the worlds quadtree leaf/loading square size, range from 2 - 12 (default = 7)
  • < 5 = small square, fast rendering
  • 7 = average square, average rendering
  • > 9 = large square, slow rendering
  • max_gpu_mem = maximum gpu memory usage in MB, 0 = auto detect (default = 0)
  • disable_cpu_saver = eihort will (true) or will not (false) continually redraw frames even nothing changes
  • worker_threads = number of worker threads to use to load the world, 0 = auto detect (default = 0)
  • ignore_gl_errors = eihort will (true) or will not (false) ignore OpenGL errors
  • silent_fail_texture_load = eihort will (true) or will not (false) complain about missing textures
developer tools:
  • => Option to control developer tools
  • deveoper_tools = eihort can (true) or can not (false) use any developer tools
  • deveoper_tools_path = the folder in which eihort will write all created files
  • print_loaded_assets = eihort will (true) or will not (false) list all loaded assests/files
  • print_found_versions = eihort will (true) or will not (false) list all known Minecraft versions
  • print_cache_hits = eihort will (true) or will not (false) list all VRAM cache hits
  • print_cache_hits_detail = eihort will (true) or will not (false) list details for VRAM cache hits
  • show_developer_buttons = eihort will (true) or will not (false) show developer buttons
  • enable_developer_keys = eihort will (true) or will not (false) enable developer keys
