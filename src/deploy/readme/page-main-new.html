  • An interactive, highly adaptable, long view-distance, 3D Minecraft world viewer which is based on OpenGL

It is perfect for taking screenshots of your massive builds, or just exploring your world. It features the most blocks of the latest Minecraft version as well as smooth lighting, biome colouring and much more.
interactive
  • change your position
  • change your spawn point
  • highlight ores
  • set up spline paths and enjoy the smoothest movements
  • control the time of day
  • control the view distance
  • take a screenshot
highly adaptable
  • use the keys you like
  • adjustable mouse sensitivity
  • load additional maps on startup
  • use your favorite texture pack
  • edite screen size and visual quality
  • change startup position on the map
  • set fps limit to save hardware capacity
long view-distance
  • up to 10'000 blocks view distance
  • adaptable view distance for slower GPUs
  • good framerate with 1'000 blocks view distance on later hardware
3D Minecraft world viewer
  • realistic 3D view on your map
  • no restrictions on your position or view angle
  • supports blocks of the latest Minecraft versions
  • supports MCRegion and Anvil maps
